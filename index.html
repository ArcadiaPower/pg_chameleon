    Welcome to the Pg Chameleon project page.

    Pg Chameleon is a MySQL to PostgreSQL replica system developed in python 2.7. The system is still under development but offers some basic services useful for replicating data from MySQL to PostgreSQL in real time.

    The system uses the python-mysql-replication to collect the data changes fromt the mysql replication protocol and use PostgreSQL as relay log and replica destination. The configuration is a simple yaml file.

    Please visit the project github page for more informations and for reporting the issues.

    The documentation is available on read the docs.
